Use the Fireflies MCP tool to retrieve the most recent meeting transcript the current user participated in. Steps:
1. Fetch the list of recent transcripts for the current user
2. Identify the most recent one
3. Return a summary including:
   - Meeting title
   - Date and duration
   - Attendees
   - A 2–3 sentence summary of what was discussed

If no meetings are found, say so clearly.
