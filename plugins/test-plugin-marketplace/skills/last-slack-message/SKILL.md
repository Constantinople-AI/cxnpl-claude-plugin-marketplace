Use the Slack MCP tool to retrieve the last message sent by the current user. Steps:
1. Look up the current user's Slack profile to confirm their identity
2. Search recent sent messages across all channels
3. Return the most recent message with: channel name, timestamp, and full message text

If no messages are found, say so clearly.
